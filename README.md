# Nnamdi | Developer Portfolio

A personal developer portfolio built from scratch with HTML, CSS, and vanilla JavaScript — no frameworks, no templates.

Live site: [nnamdi-security.github.io/portfolio](https://mynameisnnamdi.netlify.app/)

---

## About This Project

This portfolio was built as part of my #BiP60 Build in Public challenge — 60 days of learning and building in public as a trainee in the Enugu State Government technology programme run in partnership with Turing Tech LLC.

The goal was simple: build something real that represents where I am right now, and grow it as I grow.

---

## Built With

- HTML5 — semantic structure throughout
- CSS3 — custom properties, Flexbox, Grid
- Vanilla JavaScript — scroll animations, typewriter effect, hamburger menu, Intersection Observer
- GitHub Pages — hosting and deployment

---

## Features

- Fully responsive — mobile, tablet, and desktop
- Smooth scroll navigation
- Typewriter effect on the hero heading
- Scroll-triggered animations using Intersection Observer
- Downloadable resume (PDF)
- Working contact form
- Accessible markup (semantic HTML, ARIA labels)

---

## Sections

- **Home** — hero with typewriter effect and CTA buttons
- **About** — background, story, and programme context
- **Skills** — proficient stack and current learning roadmap
- **Projects** — cards with tags, links, and status indicators
- **Contact** — form and direct contact details

## Getting Started

To run this locally:

```bash
git clone https://github.com/nnamdi-security/portfolio.git
cd portfolio
```

Then open `index.html` in your browser. No build tools or dependencies required.
